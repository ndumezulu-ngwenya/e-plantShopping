# e-plantShopping
## Paradise Nursery Shopping Application 🌿

Welcome to the **Paradise Nursery** Shopping Application! This is a React-based e-commerce front-end application created for my final project. 

The application allows users to browse a variety of house plants, add them to a shopping cart, and manage their cart items before checkout.

### Features
* Browse multiple categories of house plants.
* Add plants to a dynamic shopping cart.
* Increase, decrease, or remove items from the cart.
* View total item counts and total costs in real-time.
